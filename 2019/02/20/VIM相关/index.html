下载

终端输入$ curl -fLo ~/.vim/autoload/plug.vim --create-dirs https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim

此时如果显示curl: (7) Failed to connect to 127.0.0.1 port 45997: 拒绝连接

在上述命令前加入sudo即可

即$ sudo curl -fLo ~/.vim/autoload/plug.vim --create-dirs https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim

然后重新加载进入vim，在NOMAL模式下

  • 检查状态：:PlugStatus
  • 安装插件：:PlugInstall
  • 更新插件：:PlugUpdate 按d可查看更改，或者输入:PlugDIff
  • 删除插件：首先删除或者注释掉在.vimrc中添加的plug命令，然后重启vim，再输入PlugClean
  • 升级vim-plug：PlugUpgrade

插件

goyo.vim

GitHub：https://github.com/junegunn/goyo.vim

  • 在~/.vimrc中加入Plug 'junegunn/goyo.vim'
  • 在vim中运行:PlugInstall
  • 使用：打开：:Goyo

​ 关闭：:Goyo!

limelight.vim

  • 在~/.vimrc中加入Plug 'junegunn/limelight.vim'
  • 在vim中运行:PlugInstall
  • 使用：打开：:Limelight

​ 关闭：:Limelight!

airline

GitHub：https://github.com/vim-airline/vim-airline

​ https://github.com/vim-airline/vim-airline-themes

  • 在~/.vimrc中加入Plug 'vim-airline/vim-airline-themes'和Plug 'vim-airline/vim-airline'
  • 在vim中运行:PlugInstall
  • 在~/.vimrc中加入let g:airline_theme='bubblegum'，其中bubblegum为主题名字

vim主题：vim-monokai-tasty

GitHub: https://github.com/patstockwell/vim-monokai-tasty

  • ~/.vimrc中加入Plug patstockwell/vim-monokai-tasty
  • :PlugInstall
  • 把~/.vim/plugged/vim-monokai-tasty/colors/vim-monokai-tasty.vim复制到~/.vim/colors文件夹下，在~/.vimrc中加入colorscheme vim-monokai-tasty

我在第二步时，无法clone，所以我直接在~/.vim/plugged目录下运行git clone https://github.com/patstockwell/vim-monokai-tasty.git
