Hexo创建过程中所到的问题及解决方案

安装环境为：Ubuntu1804

安装git

具体安装方法可自行百度

但是不知什么原因，我自己无法进行clone https文件，在终端显示为

fatal: unable to access 'https://github.com/jackgitgz/CnblogsSpider.git/': Failed to connect to 127.0.0.1 port 34659: 拒绝连接

后来百度之后，好像是因为我加了代理，但是在我卸载git重新安装后，仍然无法解决。然后我直接删除了.gitconfig文件，又出现了如下错误：
fatal: unable to access 'https://github.com/jackgitgz/CnblogsSpider.git/': Failed to connect to 127.0.0.1 port 45997: 拒绝连接

终端输入$ env|grep -i proxy后显示

我们需要将两个https的代理关闭，终端输入$ unset https_proxy和$ unset HTTPS_PROXY把相应的代理关掉后，就成功了。

但是每次clone之前都要先查看是否需要关闭上面两个代理

安装Hexo

在安装nodejs和npm后，输入$ sudo npm install -g hexo-cli安装Hexo。可能会出现长时间卡住问题。大概是由于网络问题。我们可以先终止下载，更改为淘宝的源。

输入sudo npm config set registry https://registry.npm.taobao.org

配置后可通过下面方式来验证是否成功 sudo npm config get registry如果显示为https://registry.npm.taobao.org/则说明成功。

然后再输入$ sudo npm install -g hexo-cli进行安装。
